# music_player

## Features

  * [x] Beautiful UI with multiple themes includes light theme and dark theme
  * [x] Play / Stop / Pause
  * [x] Seek
  * [x] Album Art
  * [x] onComplete
  * [x] onDuration / onCurrentPosition
  * [x] Fast forward / Rewind
  * [ ] Favorite
  * [ ] Shuffle
